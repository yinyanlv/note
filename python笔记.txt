字典  // 在其他语言中，字典可能会被称作关系型数组、哈希表或哈希图
dictionary  // {}，字典中元素的顺序是无关紧要的，实际存储顺序可能取决于你添加元素的顺序，字典的键必须为不可变对象，因此，元组可以作为字典的键

推导式  // 元组是没有推导式
